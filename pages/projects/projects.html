Shootout Contest

Live Demo
Code

Click on the Start button, and after short music intro, targets will appear randomly on their spots. Earned points depends on your precision. 'Green' circle is 'Bullseye', and it brings 12 points. As you progress, targets will appear faster. Every time you miss, you lose one life. Number of lives is represented by number of shiny bullets left.

Memory Game

Live Demo
Code

Hello ! This is my implementation of well known Memory Game. There are 16 cards( 8 pairs in total ). Try to reveal all the pairs with as few guesses as possible. The app is made with HTML, CSS and Vanilla JavaScript. Even being well aware that this is a fairly simple project, for the sake of practice I made it using ES6 Modules.Also, I created all of the illustrations and background used in the game.
